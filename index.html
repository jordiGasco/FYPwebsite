Academic Title:

A Data-Driven Web Application for Baseball Team Management and Performance Optimization

Commercial Title:

BaseballPro Manager

Project Description:

BaseballPro Manager is a web-based platform designed to streamline team management, player performance tracking, and lineup optimization through an intuitive and data-driven system. Built with React.js, Node.js, MongoDB, and integrated with Sportradar’s MLB API, the application provides managers with tools to analyze player statistics, generate optimal lineups, and access performance insights. Players can log in to track their own progress and receive feedback on key areas for improvement.
